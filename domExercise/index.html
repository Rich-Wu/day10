Javascript DOM

  • Selecting a single element

    You have a couple options when selecting a single element on the webpage:


    document.querySelector('selector');

    querySelector finds and selects the first element on the page that matches with the passed-in selector.


    document.getElementById('id')

    getElementById finds and selects the element on the page that has the matching ID attribute.


  • Selecting Multiple Elements

    You also have options when trying to select multiple elements in the document:


    document.querySelectorAll('selector');

    querySelectorAll finds and selects all elements in the document that match the passed-in selector.


    document.getElementsByTagName('tagName')

    getElementsByTagName finds every element that matches the specified HTML tag.


    document.getElementsByClassName('className')

    getElementsByClassName finds every element that matches the specified HTML class.
